# Stopwatch
Stopwatch based on one of [Florin Pop app ideas](https://github.com/florinpop17/app-ideas/blob/master/Projects/1-Beginner/Stopwatch-App.md)

## Features
- Start timer
- Stop timer
- Log laps
- Reset timer and laps
- Restart timer (only if timer is started)
